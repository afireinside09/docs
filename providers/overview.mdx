## Choosing a Provider

### For General Text

All providers offer high-quality text generation. Consider:

| If You Need | Recommended |
|-------------|-------------|
| Best overall quality | OpenAI GPT-4o, Claude 3.5 Sonnet |
| Fastest responses | OpenAI GPT-4o-mini, Gemini Flash |
| Complex reasoning | Claude 3.5 Sonnet, OpenAI o3 |
| Lowest cost | OpenAI GPT-4o-mini, Straico |
| Model variety | OpenRouter |

### For Images

| Provider | Models | Best For |
|----------|--------|----------|
| OpenAI | DALL-E 3, GPT-Image-1 | High quality, consistent style |
| Gemini | Gemini Flash Image | Google ecosystem |
| OpenRouter | Multiple | Access to various models |
| Straico | Multiple | Simple pricing |

### For Audio

Only OpenAI supports text-to-speech:
- 11 voice options
- High-fidelity output
- Multiple languages

### For Video

| Provider | Models | Notes |
|----------|--------|-------|
| OpenAI | Sora 2, Sora 2 Pro | Requires organization verification |
| Gemini | Veo 3, Veo 3 Fast | Google ecosystem |

## Setting Up a Provider

<Steps>
  <Step title="Get API Key">
    Create an account with your chosen provider and generate an API key.
  </Step>
  <Step title="Add to SheetMagic">
    Go to **Extensions > SheetMagic > Manage AI API Keys** and add your key.
  </Step>
  <Step title="Select Provider">
    Open **Extensions > SheetMagic > Open AI Settings** and select your provider.
  </Step>
  <Step title="Choose Model">
    Select your preferred model from the dropdown.
  </Step>
</Steps>

## API Key Security

<Warning>
Never share your API keys. SheetMagic stores keys securely in your Google account's script properties, encrypted and accessible only to you.
</Warning>

Best practices:
- Use separate API keys for SheetMagic
- Set spending limits with your provider
- Rotate keys periodically
- Monitor usage in your provider dashboard

## Switching Providers

You can switch providers at any time:

1. Add the new provider's API key
2. Select the new provider in settings
3. Choose a model

Existing formulas will use the new provider on their next execution.

## Rate Limits

Each provider has different rate limits:

| Provider | Typical Limits |
|----------|---------------|
| OpenAI | Varies by tier |
| Gemini | Varies by tier |
| Claude | Varies by tier |
| OpenRouter | Pooled limits |
| Straico | Based on plan |

SheetMagic handles rate limiting automatically by queuing requests and retrying when limits are exceeded.
