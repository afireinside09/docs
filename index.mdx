## What is SheetMagic?

SheetMagic is a Google Sheets add-on that brings the power of AI directly into your spreadsheets. Use simple formulas to generate content, analyze data, create images, produce audio, and extract information from the web.

## Quick Start

Get started with SheetMagic in three steps:

<Steps>
  <Step title="Install the Add-on">
    Install SheetMagic from the [Google Workspace Marketplace](https://workspace.google.com/marketplace).
  </Step>
  <Step title="Configure Your API Key">
    Open **Extensions > SheetMagic > Open AI Settings** and add your API key for your preferred provider.
  </Step>
  <Step title="Start Using AI Functions">
    Type `=AITEXT("Your prompt here")` in any cell to generate AI content.
  </Step>
</Steps>

## Available Functions

### AI Functions

| Function | Description |
|----------|-------------|
| `=AITEXT(prompt)` | Generate text content |
| `=AILIST(prompt)` | Generate a vertical list |
| `=AILISTH(prompt)` | Generate a horizontal list |
| `=AIIMAGE(description)` | Generate images |
| `=AISPEECH(text, voice)` | Convert text to speech |
| `=AIVIDEO(description)` | Generate videos |
| `=GPTV(imageUrl, prompt)` | Analyze images with AI vision |

### Web Scraping Functions

| Function | Description |
|----------|-------------|
| `=VISIT(url)` | Extract clean text from a webpage |
| `=SERP(query)` | Search the web and return results |
| `=PAGEDATA(urls)` | Get metadata for multiple URLs |
| `=GETSELECTOR(url, selector)` | Extract elements using CSS selectors |
| `=GETIMG(url)` | Extract all image URLs from a page |
| `=DOMAINCHECK(domain)` | Check domain availability |
