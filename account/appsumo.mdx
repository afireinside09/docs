## About AppSumo Codes

SheetMagic lifetime deals are available through [AppSumo](https://appsumo.com). When you purchase a deal, you receive a redemption code that grants lifetime access to SheetMagic.

## Redeeming Your Code

<Steps>
  <Step title="Log In">
    Go to [sheetmagic.ai](https://sheetmagic.ai) and log in or create an account.
  </Step>
  <Step title="Navigate to Redeem">
    Go to your dashboard and find the AppSumo Redemption section.
  </Step>
  <Step title="Enter Code">
    Paste your AppSumo code exactly as provided.
  </Step>
  <Step title="Click Redeem">
    Click the Redeem button to activate your license.
  </Step>
  <Step title="Confirm">
    You'll see a success message and your new license details.
  </Step>
</Steps>

## What You Get

When you redeem an AppSumo code:

| Feature | Details |
|---------|---------|
| Duration | Lifetime (never expires) |
| Seats | 1 per code |
| AI Functions | Full access |
| Web Scraping | Full access |
| Updates | Included forever |
| Support | Standard support |

## Stacking Codes

If you purchased multiple AppSumo codes:
- Each code creates a separate license
- Codes cannot be combined into one license
- Each license includes 1 seat

<Note>
Multiple codes give you multiple separate licenses, not additional seats on one license.
</Note>

## Before Redeeming

### Validate Your Code

You can check if a code is valid before redeeming:

1. Go to the AppSumo Redemption page
2. Enter your code
3. Click **Validate** (if available)
4. See code status and details

### Code Status

| Status | Meaning |
|--------|---------|
| Available | Code is valid and ready to redeem |
| Redeemed | Code has already been used |
| Expired | Code is past its expiration date |
| Invalid | Code is not recognized |

## After Redemption

Once redeemed:

1. Your license key is generated automatically
2. Find it in your dashboard under Licenses
3. Use this key in Google Sheets to activate SheetMagic

### Activating in Google Sheets

1. Open Google Sheets
2. Go to **Extensions > SheetMagic > Start SheetMagic**
3. Enter your license key (from dashboard)
4. Enter your account email
5. Click Activate

## Troubleshooting

<AccordionGroup>
  <Accordion title="Code says 'already redeemed'">
    Each AppSumo code can only be redeemed once. If you believe this is an error:
    - Check if you previously redeemed it on another account
    - Contact support with your order details
  </Accordion>
  <Accordion title="Code says 'invalid'">
    - Double-check the code for typos
    - Ensure there are no extra spaces
    - Verify it's a SheetMagic code (not for another product)
  </Accordion>
  <Accordion title="Code says 'expired'">
    AppSumo codes may have expiration dates. If your code expired:
    - Contact AppSumo support for assistance
    - Check your AppSumo account for updated codes
  </Accordion>
  <Accordion title="Can't find my license after redemption">
    - Log in to sheetmagic.ai
    - Go to Dashboard > Licenses
    - Your AppSumo license should appear there
    - If not, contact support with your AppSumo order ID
  </Accordion>
  <Accordion title="Need more seats">
    AppSumo codes provide 1 seat each. For multiple team members:
    - Purchase additional AppSumo codes (if available)
    - Purchase additional licenses from sheetmagic.ai
  </Accordion>
</AccordionGroup>

## Refunds

AppSumo purchases are subject to AppSumo's refund policy:
- Refund requests go through AppSumo
- Once redeemed, codes cannot be transferred
- Contact AppSumo support for refund questions
