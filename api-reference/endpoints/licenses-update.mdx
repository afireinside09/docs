---
title: "Update License"
api: "PATCH /licenses/{id}"
description: "Update license status or seat count"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

<ParamField path="id" type="string" required>
  License ID (UUID)
</ParamField>

<ParamField body="status" type="string">
  New status: `active`, `cancelled`, `suspended`
</ParamField>

<ParamField body="seat_count" type="number">
  New seat count
</ParamField>

## Validation Rules

- Cannot reduce `seat_count` below currently active seats
- Lifetime licenses only support 1 seat

## Response

<ResponseField name="license" type="object">
  The updated license object
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X PATCH "https://api.sheetmagic.ai/licenses/123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "seat_count": 10
  }'
```

```javascript JavaScript
const response = await fetch('https://api.sheetmagic.ai/licenses/123e4567-e89b-12d3-a456-426614174000', {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    seat_count: 10
  })
});
const data = await response.json();
```
</CodeGroup>
