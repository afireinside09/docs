---
title: "Get License"
api: "GET /licenses/{id}"
description: "Retrieve a specific license by ID"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

<ParamField path="id" type="string" required>
  License ID (UUID)
</ParamField>

## Response

<ResponseField name="license" type="object">
  The license object
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.sheetmagic.ai/licenses/123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.sheetmagic.ai/licenses/123e4567-e89b-12d3-a456-426614174000', {
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  }
});
const data = await response.json();
```
</CodeGroup>

## Errors

| Status | Description |
|--------|-------------|
| 401 | Invalid or missing token |
| 404 | License not found |
| 500 | Server error |

<Note>
Users can only access their own licenses. Admins can access any license.
</Note>
