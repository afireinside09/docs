---
title: "List Seats"
api: "GET /licenses/{licenseId}/seats"
description: "List all seats for a license"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

<ParamField path="licenseId" type="string" required>
  License ID (UUID)
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `active` or `revoked`
</ParamField>

<ParamField query="limit" type="number" default="50">
  Maximum seats to return
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of seats to skip
</ParamField>

## Response

<ResponseField name="seats" type="array">
  Array of seat objects
</ResponseField>

<ResponseField name="count" type="number">
  Total seats matching filter
</ResponseField>

<ResponseField name="active_count" type="number">
  Number of active seats
</ResponseField>

<ResponseField name="available_slots" type="number">
  Remaining seats available
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.sheetmagic.ai/licenses/123e4567-e89b-12d3-a456-426614174000/seats?status=active" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch(

  {
```
