---
title: "Transfer Seat"
api: "POST /licenses/{licenseId}/seats/{seatId}/transfer"
description: "Transfer a seat to a different email address"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

<ParamField path="licenseId" type="string" required>
  License ID (UUID)
</ParamField>

<ParamField path="seatId" type="string" required>
  Seat ID (UUID)
</ParamField>

<ParamField body="new_email" type="string" required>
  New email address for the seat
</ParamField>

## Validation

- Seat must be active
- New email must not already have a seat (active or revoked)
- New email cannot be the same as current email

## Response

<ResponseField name="seat" type="object">
  The updated seat object
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X POST "https://api.sheetmagic.ai/licenses/123e4567-e89b-12d3-a456-426614174000/seats/456e7890-a12b-34c5-d678-901234567890/transfer" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "new_email": "newemail@example.com"
  }'
```

```javascript JavaScript
const response = await fetch(
  'https://api.sheetmagic.ai/licenses/123e4567-e89b-12d3-a456-426614174000/seats/456e7890-a12b-34c5-d678-901234567890/transfer',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer <token>',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      new_email: 'newemail@example.com'
    })
  }
);
const data = await response.json();
```
</CodeGroup>

## Response Example

```json
{
  "seat": {
    "id": "456e7890-a12b-34c5-d678-901234567890",
    "email": "newemail@example.com",
    "status": "active",
    "created_at": "2024-06-01T00:00:00Z",
    "license_id": "123e4567-e89b-12d3-a456-426614174000"
  },
  "message": "Seat transferred from oldemail@example.com to newemail@example.com"

```

## Errors

| Status | Description |
|--------|-------------|
| 400 | Seat inactive or same email |
| 401 | Invalid or missing token |
| 404 | License or seat not found |
| 409 | New email already has a seat |
| 500 | Server error |
