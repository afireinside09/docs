---
title: "Create Checkout Session"
api: "POST /payments/checkout"
description: "Create a Stripe checkout session for license purchase"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

<ParamField body="seat_count" type="number" required>
  Number of seats for the license
</ParamField>

<ParamField body="duration" type="string" required>
  License duration: `1_month`, `1_year`, or `lifetime`
</ParamField>

<ParamField body="success_url" type="string">
  URL to redirect after successful payment
</ParamField>

<ParamField body="cancel_url" type="string">
  URL to redirect if payment is cancelled
</ParamField>

<ParamField body="client_idempotency_key" type="string">
  Optional key to prevent duplicate sessions
</ParamField>

## Response

<ResponseField name="session_id" type="string">
  Stripe checkout session ID
</ResponseField>

<ResponseField name="url" type="string">
  Checkout page URL
</ResponseField>

<ResponseField name="expires_at" type="string">
  Session expiration timestamp
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X POST "https://api.sheetmagic.ai/payments/checkout" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "seat_count": 5,
    "duration": "1_year",
    "success_url": "https://sheetmagic.ai/success",
    "cancel_url": "https://sheetmagic.ai/pricing"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.sheetmagic.ai/payments/checkout', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    seat_count: 5,
    duration: '1_year',
    success_url: 'https://sheetmagic.ai/success',
    cancel_url: 'https://sheetmagic.ai/pricing'
  })
});
const data = await response.json();

// Redirect to checkout
window.location.href = data.url;
```
</CodeGroup>

## Response Example

```json
{
  "session_id": "cs_live_abc123def456",
  "url": "https://checkout.stripe.com/c/pay/cs_live_abc123def456",
  "expires_at": "2024-12-04T13:00:00Z"

```

## Integration Flow

1. Call this endpoint to create a session
2. Redirect user to the `url`
3. User completes payment on Stripe
4. Stripe redirects to your `success_url`
5. Webhook creates the license automatically
