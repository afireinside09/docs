---
title: "Create License"
api: "POST /licenses"
description: "Create a new license"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

<ParamField body="seat_count" type="number" required>
  Number of seats for the license
</ParamField>

<ParamField body="duration" type="string" required>
  License duration: `1_month`, `1_year`, or `lifetime`
</ParamField>

<Note>
Lifetime licenses only support 1 seat. Attempting to create a lifetime license with multiple seats will fail.
</Note>

## Response

<ResponseField name="license" type="object">
  The created license object
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X POST "https://api.sheetmagic.ai/licenses" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "seat_count": 5,
    "duration": "1_year"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.sheetmagic.ai/licenses', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    seat_count: 5,
    duration: '1_year'
  })
});
const data = await response.json();
```
</CodeGroup>

## Errors

| Status | Description |
|--------|-------------|
| 400 | Invalid parameters (e.g., lifetime with multiple seats) |
| 401 | Invalid or missing token |
| 500 | Server error |
