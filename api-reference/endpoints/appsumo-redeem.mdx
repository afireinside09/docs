---
title: "Redeem AppSumo Code"
api: "POST /appsumo/redeem"
description: "Redeem an AppSumo code for a lifetime license"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

<ParamField body="code" type="string" required>
  AppSumo redemption code
</ParamField>

## Response

<ResponseField name="result" type="object">
  <Expandable title="properties">
    <ResponseField name="success" type="boolean">
      Whether redemption succeeded
    </ResponseField>
    <ResponseField name="license_id" type="string">
      Created license ID
    </ResponseField>
    <ResponseField name="license" type="object">
      Full license object
    </ResponseField>
    <ResponseField name="message" type="string">
      Status message
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X POST "https://api.sheetmagic.ai/appsumo/redeem" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "APPSUMO-XXXX-XXXX"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.sheetmagic.ai/appsumo/redeem', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    code: 'APPSUMO-XXXX-XXXX'
  })
});
const data = await response.json();
```
</CodeGroup>

## Error Response

```json
{
  "error": "Redemption failed",
  "message": "Code has already been redeemed",
  "error_code": "ALREADY_REDEEMED"
}
```

## Error Codes

| Code | Description |
|------|-------------|
| `INVALID_CODE` | Code not recognized |
| `ALREADY_REDEEMED` | Code already used |
| `EXPIRED` | Code past expiration |
| `UNKNOWN_ERROR` | Other failure |

## Errors

| Status | Description |
|--------|-------------|
| 400 | Invalid or unavailable code |
| 401 | Invalid or missing token |
| 500 | Server error |
