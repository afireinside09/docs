---
title: "Get Current User"
api: "GET /users/me"
description: "Retrieve the authenticated user's profile"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

## Response

<ResponseField name="user" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Unique user identifier (UUID)
    </ResponseField>
    <ResponseField name="email" type="string">
      User's email address
    </ResponseField>
    <ResponseField name="full_name" type="string">
      User's display name (nullable)
    </ResponseField>
    <ResponseField name="is_admin" type="boolean">
      Whether user has admin privileges
    </ResponseField>
    <ResponseField name="stripe_customer_id" type="string">
      Associated Stripe customer ID (nullable)
    </ResponseField>
    <ResponseField name="created_at" type="string">
      Account creation timestamp
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      Last update timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.sheetmagic.ai/users/me" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.sheetmagic.ai/users/me', {
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  }
});
const data = await response.json();
```
</CodeGroup>

## Errors

| Status | Description |
|--------|-------------|
| 401 | Invalid or missing token |
| 404 | User profile not found |
| 500 | Server error |
