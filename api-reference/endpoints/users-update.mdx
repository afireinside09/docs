---
title: "Update Current User"
api: "PATCH /users/me"
description: "Update the authenticated user's profile"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

<ParamField body="full_name" type="string">
  Updated display name
</ParamField>

## Response

<ResponseField name="user" type="object">
  Updated user object (same structure as GET /users/me)
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X PATCH "https://api.sheetmagic.ai/users/me" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"full_name": "Jane Doe"}'
```

```javascript JavaScript
const response = await fetch('https://api.sheetmagic.ai/users/me', {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    full_name: 'Jane Doe'
  })
});
const data = await response.json();
```
</CodeGroup>
