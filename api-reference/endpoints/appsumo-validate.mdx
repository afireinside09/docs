---
title: "Validate AppSumo Code"
api: "POST /appsumo/validate"
description: "Check if an AppSumo code is valid before redemption"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth
</ParamField>

<ParamField body="code" type="string" required>
  AppSumo code to validate
</ParamField>

## Response

<ResponseField name="valid" type="boolean">
  Whether code can be redeemed
</ResponseField>

<ResponseField name="message" type="string">
  Status description
</ResponseField>

<ResponseField name="status" type="string">
  Code status: `available`, `redeemed`, `expired`
</ResponseField>

<ResponseField name="expires_at" type="string">
  Code expiration date (nullable)
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X POST "https://api.sheetmagic.ai/appsumo/validate" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "APPSUMO-XXXX-XXXX"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.sheetmagic.ai/appsumo/validate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    code: 'APPSUMO-XXXX-XXXX'
  })
});
const data = await response.json();
```
</CodeGroup>

## Valid Code Response

```json
{
  "valid": true,
  "message": "Code is valid and available for redemption",
  "status": "available",
  "expires_at": "2025-12-31T23:59:59Z"
}
```

## Invalid Code Responses

### Already Redeemed

```json
{
  "valid": false,
  "message": "This code has already been redeemed",
  "status": "redeemed",
  "expires_at": null
}
```

### Expired

```json
{
  "valid": false,
  "message": "This code has expired",
  "status": "expired",
  "expires_at": "2024-01-01T00:00:00Z"
}
```

### Invalid

```json
{
  "valid": false,
  "message": "Invalid redemption code",
  "status": null,
  "expires_at": null
}
```

## Use Case

Use this endpoint to check code validity before attempting redemption:

```javascript
// First validate
const validateResponse = await fetch('/appsumo/validate', {
  method: 'POST',
  body: JSON.stringify({ code: userCode })
});
const validation = await validateResponse.json();

if (validation.valid) {
  // Then redeem
  const redeemResponse = await fetch('/appsumo/redeem', {
    method: 'POST',
    body: JSON.stringify({ code: userCode })
  });
  // Handle success
} else {
  // Show error to user
  alert(validation.message);
}
```
