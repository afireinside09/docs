## Overview

The SheetMagic API provides programmatic access to license management, seat administration, payment processing, and AppSumo code redemption.

<Note>
This API is primarily used internally by the SheetMagic add-on and dashboard. Most users won't need to interact with it directly.
</Note>

## Base URL

```
https://api.sheetmagic.ai
```

## Response Format

All responses are JSON formatted:

```json
{
  "data": { ... },
  "error": null
}
```

Error responses:

```json
{
  "error": "Error type",
  "message": "Human-readable description",
  "error_code": "SPECIFIC_ERROR_CODE"
}
```

## HTTP Status Codes

| Code | Description |
|------|-------------|
| 200 | Success |
| 201 | Created |
| 204 | No Content (successful deletion) |
| 400 | Bad Request |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
| 409 | Conflict |
| 500 | Internal Server Error |

## Rate Limiting

| Endpoint Type | Limit |
|---------------|-------|
| Standard | Varies by tier |
| License Verification | 20 requests/minute per IP |

Rate limit headers:
- `X-RateLimit-Limit`: Maximum requests
- `X-RateLimit-Remaining`: Remaining requests
- `X-RateLimit-Reset`: Reset timestamp

## Pagination

List endpoints support pagination:

```
GET /licenses?limit=50&offset=0
```

| Parameter | Description | Default |
|-----------|-------------|---------|
| `limit` | Maximum items to return | 50 |
| `offset` | Number of items to skip | 0 |

Response includes count:

```json
{
  "licenses": [...],
  "count": 150
}
```

## Filtering

Many list endpoints support filtering:

```
GET /licenses?status=active&type=monthly
```

Available filters vary by endpoint.

## Health Check

Check API availability:

```bash
GET /health
```

Response:
```json
{
  "status": "ok",
  "timestamp": "2024-12-04T12:00:00Z",
  "uptime": 3600.5,
  "environment": "production",
  "database": "connected"
}
```

## SDK & Libraries

Currently, the API is used directly via HTTP. No official SDK is provided.

### Example with cURL

```bash
curl -X GET "https://api.sheetmagic.ai/health" \
  -H "Content-Type: application/json"
```

### Example with JavaScript

```javascript
const response = await fetch('https://api.sheetmagic.ai/health');
const data = await response.json();
```
